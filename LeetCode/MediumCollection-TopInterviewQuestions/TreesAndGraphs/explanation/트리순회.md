

# PreOrder - 전위 순회
1. 노드를 방문한다
2. **왼쪽**서브트리를 전위 순회한다.
3. **오른쪽** 서브트리를 전위 순회 한다.
* 전위 순회는 깊이 우선순회(DFS)라고도 한다

# InOrder - 중위 순회
1. 왼쪽 서브트리를 중위 순회한다.
2. 노드를 방문한다.
3. 오른쪽 서브트리를 중위 순회한다
* 대칭순회(Symmetric)라고도 한다

# PostOrder - 후위 순회
1. 왼쪽 서브트리를 후위순회한다
2. 오른쪽 서브트리를 후위순회 한다.
3. 노드를 방문한다.

# Breadth Frist - 깊이(레벨) 순서 순회
1. 모든 노드를 낮은 레벨 부터 차례대로 순회한다
2. 각 레벨의 **왼쪽** 노드 값 부터 순회한다..
